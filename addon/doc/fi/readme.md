# Sovellussanasto

## 

## Tiedot


* Tekijät: Rui Fontes ja Ricardo M. Leonarczyk yhteistyössä Cyrille Bougot'n kanssa
* Yhteensopivuus: NVDA 2023.1 ja uudemmat



## Käyttö

Tämän yksinkertaisen lisäosan avulla on mahdollista luoda NVDA:lle sovelluskohtaisia sanastoja.
Käytä lisäosaa siirtymällä sovellukseen, jolle haluat luoda sanaston, ja paina NVDA+Vaihto+Ctrl+D.
Tämän jälkeen avautuu tavallinen NVDA:n sanastovalintaikkuna, johon voit lisätä vain aktiivisessa sovelluksessa käytettäviä määrityksiä (sovelluksen nimi näkyy valintaikkunan otsikossa).
Oletuspikanäppäintä voidaan vaihtaa NVDA:n Näppäinkomennot-valintaikkunan Asetukset-kategorian kohdassa "Näyttää sovelluskohtaisen sanastovalintaikkunan".



## Näppäinkomento

NVDA+Ctrl+Vaihto+D
