# \#Sovellussanasto

## 

## 

## \##Tiedot

* Tekijät: Rui Fontes ja Ricardo M. Leonarczyk yhteistyössä Cyrille Bougot'n kanssa
* Yhteensopivuus: NVDA 2023.1 ja uudemmat



## \##Käyttö

Tämä yksinkertainen lisäosa mahdollistaa sovelluskohtaisten sanastojen luomisen NVDA:lle.
Käytä lisäosaa siirtymällä sovellukseen, jolle haluat sanaston, ja paina NVDA+Vaihto+Ctrl+D.
Tämän jälkeen avautuu tavallinen NVDA:n sanastovalintaikkuna, johon voit lisätä määrityksiä, jotka ovat voimassa vain aktiivisessa sovelluksessa (sovelluksen nimi näkyy valintaikkunan otsikossa).
Oletuspikanäppäintä on mahdollista muuttaa NVDA:n Näppäinkomennot-valintaikkunan Asetukset-kategorian kohdassa "Näyttää sovelluskohtaisen sanastovalintaikkunan".



## \##Komento

NVDA+Vaihto+Ctrl+D
